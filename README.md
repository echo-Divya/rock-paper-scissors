# Interactive Rock Paper Scissors Game

A classic Rock Paper Scissors game built with modern, modular web technologies. This project focuses on pure JavaScript logic, dynamic DOM manipulation, and state tracking.

## ✨ Features
* **State Management:** Tracks wins, losses, and ties persistently across rounds using a central JavaScript object.
* **Modern Event Architecture:** Built using clean execution loops with `addEventListener` hooks rather than inline scripts.
* **Dynamic UI Updates:** Manipulates DOM text elements and toggles flex layouts conditionally based on game states.

## 🛠️ Technologies Used
* HTML5 (Semantic Structure)
* CSS3 (Custom Styling & Hover Transitions)
* JavaScript (ES6+ Logic, Conditional Algorithms)
